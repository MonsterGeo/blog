添加过程噪声

在这个章节，我们将在之前说的一维卡尔曼滤波器中添加过程噪声并分析

完整一维卡尔曼滤波模型

过程噪声

我们使用q表示过程噪声，协方差外推方程应该包含我们的过程噪声，因为这个噪声对误差是无法避免的。因此恒定动态模型的协方差外推方程为：

$$ \begin{aligned} p_{n+1,n} = p_{n,n} + q_n \end{aligned} $$

我们把这个公式加入到我们总结的一维卡尔曼滤波表格

现在我们使用一个例子。

测量箱中的水温

我们假设在稳态下，液体温度是恒温的，但真实的液体也许会有一点可能，我们用下面方程描述系统的动态特性：

xn = T + wn

其中T是稳定温度，wn是关于随机变量q的噪声。

现在，我们做些假设

  • 我们假设真实温度是50摄氏度
  • 假设模型是准确的，因此我们将过程噪声q设为极小值0.0001
  • 误差在0.1摄氏度
  • 每5秒测量一次
  • 每次的实际温度为: 50.005, 49.994, 49.993, 50.001, 50.006, 49.998, 50.021, 50.005, 50 , 49.997
  • 每次测量值为：49.986, 49.963, 50.9, 50.001, 50.018, 50.05, 49.938, 49.858, 49.965, 50.114

依旧简单初始化，我们观测到温度大概在60度，因此x̂0, 0 = 60，

现在，我们假设侧脸误差在100，那么方差就是p0, 0 = 1002。

由于这个方差非常高，如果用接近实际值的值来做初始化收敛就会更高，现在我们开始测量

这是个静态的系统，那么第一次的温度就是x̂1, 0 = 60，方差为p1, 0 = p0, 0 + q = 10000 + 0.0001 = 10000.0001

对第一次测量，z1 = 49.986，那么测量误差为$ 0.1，那么r_1 = 0.01$。现在我们计算卡尔曼增益

$$ \begin{aligned} K_1 = \frac{p_{1,0}}{p_{1,0}+r_1} = 0.999... \end{aligned} $$

那么第一次估计值如下

$$ \begin{aligned} \hat x_{1,1} = \hat{1,0} +K_1(z_1 - \hat x_{1,0}) = 49.986 \end{aligned} $$

更新方差为p1, 1 = (1 − K1)p1, 0 = 0.01

那么p2, 1 = 0.01 + q = 0.0101
