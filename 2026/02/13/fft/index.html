This article will display how fft does work. If you learn algebra before, you will join this article. It will fun so much. basically in this article, code is simple, most part is related to math.

Let’s start at Polynomial.

basic definitions

defined(Polynomial): A polynomial f(x) in the variable x is defined on field of algebra F. it looks like algebraic sum of the variable x with coefficients ai:

$$ f(x) = \sum^{n-1}_{i = 0} a_ix^i $$

with a0, a1, ⋯, an − 1 is the cofficients on F

If ak is leading non-zero coefficient of the polynomial f(x). we say the highest degree of f(x) is k，denoted by d(f(x)) = k

Based on this, we can define some operations on polynomials.

defintion(addition): Given degree of n polynomial f(x), g(x)

$$ \begin{array}{lll} f(x) &=& \sum^{n}_{i=0} a_ix^i\\ g(x) &=& \sum^n_{i=0} b_ix^i \end{array} $$

with ai, bj is coefficient of field F, then defined addition of f(x), g(x) is f(x) + g(x) be

$$ f(x) + g(x) = \sum^n_{i = 0} (a_i+b_i)x^i $$

we can quick prove the d(f(x) + g(x)) = n

defintion(multiplication):Given polynomial f(x), g(x) of degree n , as define in addition. we define multiplication of polynomial donoted f(x) ⋅ g(x):

$$ \begin{aligned} f(x)\cdot g(x) =& \left(\sum^n_{i=0}a_ix^i\right) \cdot \left(\sum^n_{i=0}b_ix^i\right)\\ =&\sum^{2n}_{k = 0}c_kx^k \end{aligned} $$

where $c_k = \sum^k_{i=0}a_ib_{k-i}$

Another representation of the polynomial

Another way to write it is use vector to rewrite polynomial

Given poly f(x) as define in addition,then we can write it be

$$ f(x) = \sum^n_{i=0} a_ix^i = (a_0,a_1,\cdots,a_n) $$

Point-value Representaion

This is important part of fft. we will after explaining.

Given degree n of polynomal, the point-value representation is set consisting of n point-value pairs:

{(x0, y0), (x1, y1), ⋯, (xn, yn)}

where for k = 0, 1, ⋯, n ,yk = A(xk) and all x_k are not equal to each other.

Theorem(Uniqueness of Interpolation) For any set consisting of n point-value pairs{(x0, y0), (x1, y1), ⋯, (xn, yn)}, all xk are not equal to each other; There exist a unique degree n of polynomalf(x) satisfyyk = f(xk), k = 0, 1, ⋯, n − 1

It’s easy to prove, we talk about why we need this. the poly multiplication is very complex, regular method wasted so many time.so we need find another way. In another way, we just take multiplication be few addition or more. Frist we use Interpolation, take point x1, x2，attention the pointf(x1) * f(x2) = y1 * y2, that mean we don’t need calculate the polynomal mult, we just calculate the y1then calculate y2，It’s easy than that. and then we calcuate the pointy1 * y2，that’s enought.

For example , f(x) = x2 + 1, g(x) = x + 1, then f(1) = 2, f(2) = 5, g(1) = 2, g(2) = 3，if we calcuate f(1) * g(1) like this

f(1) * g(1) = (x2 + 1)(x + 1) = x3 + x + x2 + 1 = 4

It was very complex, but if we calcuate f(1) = 2, g(1) = 2，attention f(1) * f(2) = 2 * 2 = 4，but we don’t know the new polyf(x)g(x) is it unique? so we need prove this. if unique, we can quick get the 2 point mult result of polynomal.

Now we prove this theorem:

Proof: Recall the vector {(x0, y0), (x1, y1), ⋯, (xn, yn)} how’d it get here. we expand all point form vector then

$$ \begin{bmatrix} 1 & x_0 & x_0^2 & \dots & x_0^{n-1} \\ 1 & x_1 & x_1^2 & \dots & x_1^{n-1} \\ \vdots & \vdots & \vdots & \ddots & \vdots \\ 1 & x_{n-1} & x_{n-1}^2 & \dots & x_{n-1}^{n-1} \end{bmatrix} \begin{bmatrix} a_0 \\ a_1 \\ \vdots \\ a_{n-1} \end{bmatrix} = \begin{bmatrix} y_0 \\ y_1 \\ \vdots \\ y_{n-1} \end{bmatrix} $$

DFT and FFT

At this part, we research the property of complex roots of unity

we say, If use unit root to implement algorithm can let time complexity from Θ(n2) to Θ(nlg n), in this page, we given the defintion of unit roots, DFT. Then we explanation FFT how to compute DFT and inv of DFT in complexity time of Θ(nlg n)

complex roots of unity

the root of unity are satisfy ωn = 1 , where ω is Complex number. nth unit root have n’s. we can prove for k = 0, 1, ⋯, n − 1,this root is form ω = e2πik/n

If you learn abstract algebra, you must be familiar with multiplicative group. Yep, the root of unity form the multiplicative group. Use the modular properties, we have

ωiωj = ω(i + j) mod  n

we stop talk more properties of unit root. you can find it at any algebra books.

now we take some theorem of unit root

Theorem( Cancellation Lemma): for any n ≥ 0, k ≥ 0, d > 0 wheren, k, d ∈ ℤ we have

ωdndk = ωnk

Proof:

ωdndk = (e2πi/dn)dk = (e2πi/n)k = ωnk

corollary: extend Cancellation Lemma, for any even number n > 0,we have

ωnn/2 = ω2 = −1

Proof

ωnn/2 = e2πi(n/2)/n = eiπ = −1

and then

ω2 = e2πi/2 = eiπ = −1

Now we see another theorem

Lemma(The Halving Lemma) if n > 0 is even, The set of squares of the n-th roots of unity is The set of squares of the n/2-th roots of unity.

Proof: For Cancellation Lemma for any non-negative numberk， we have (ωnk)2 = ωn/2k. we square for all nth root of unity. have

(ωnk + n/2)2 = ωn2k + n = ωn2kωnn = ωn2k = (ωnk)2

Therefore, ωnk the same as ωk + n/2. Another easy prove is use the above corollary, ωnn/2 = −1, then ωnk + n/2 = −ωnk ,so (ωk + n/2)2 = (ωnk)2

The Summation Lemma: For any n ≥ 1 ∈ ℤ and can’t divided by n non-negative k ∈ ℤ, we have

$$ \sum^{n-1}_{j=0}(\omega^k_n)^j = 0 $$

Proof:

$$ \sum^{n-1}_{j=0} (\omega_n^k)^j = \frac{(\omega^k_n)^n - 1}{\omega^k_n - 1} = \frac{(\omega^n_{n})^k - 1}{\omega^k_n -1} = \frac{1^k -1}{\omega^k_n - 1} = 0 $$

ωnk = 1 if and only if k ∣ n. And the denominator is not zero.

For poly A(x), use even indices and odd indices, rewrite two new poly of degree n/2 A[0](x) and A[1](x):

$$ \begin{aligned} A^{[0]}(x) = a_0 + a_2x+a_4x^2+\cdots + a_{n-2}x^{n/2 -1}\\ A^{[1]}(x) = a_1 + a_3x+a_5x^2+\cdots + a_{n-1}x^{n/2 -1}\\ \end{aligned} $$

iFFT(The inverse of FFT)

Now we have the FFT. It’s time to consider how to get iFFT(FFT of inv) from FFT.

Recall the formula.The α = (a0, ⋯, an − 1) of FFT do this:

$$ y_k = \text{FFT}(\alpha) = \sum^{n-1}_{i=0} a_i * \omega^{kj}_n $$

Then, for y = (y0, ⋯, yn − 1)we rewrite this formula to form

$$ \text{iFFT}(y) = \sum^{n-1}_{k = 0}\left(\sum^{n-1}_{i=0} a_i * \omega^{kj}_n\right)\omega^{-kn}_n $$

Actually, That’s the expanded form with inverse of Vandermonde matrix.

Note the root of unity have Summation Lemma. for any root each not equal other. They sum equal zero. so

$$ \text{iFFT}(y) = \sum^{n-1}_{k = 0}\left( \sum^{n-1}_{i=0}a_i(\omega^{k(j-n)}_n) \right) $$

when j ≠ n ,They sum equal zero, so, the sum of inside remind the ωnk(j − n),wherej = n.ωnk * 0 = 1,Therefore the sum be $$ \sum^{n-1}_{k=0}\sum^{n-1}_{i=0}a_i = n*\alpha $$

we have

iFFT(yk) = n ⋅ α
