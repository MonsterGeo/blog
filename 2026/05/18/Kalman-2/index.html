这节是续上上一节的内容，我们将使用一个三阶滤波器α − β − γ滤波器来追踪加速飞行的航空器

使用α − β − γ滤波器

α − β − γ滤波器考虑了目标的加速度，因此，状态预测方程变为

$$\hat{x}_{n+1,n} = \hat{x}_{n,n} + \hat{\dot{x}}_{n,n} \Delta t + \hat{\ddot{x}}_{n,n} \frac{\Delta t^2}{2}$$

$$\hat{\dot{x}}_{n+1,n} = \hat{\dot{x}}_{n,n} + \hat{\ddot{x}}_{n,n} \Delta t$$

$$\hat{\ddot{x}}_{n+1,n} = \hat{\ddot{x}}_{n,n}$$

其中$\ddot{x}_{n}$是加速度（x的二阶导）

那么更新方程如下描述

$$ \begin{aligned} \hat{x}_{n,n} = \hat{x}_{n,n-1} + \alpha(z_n - \hat{x}_{n,n-1}) \end{aligned} $$

$$\hat{\dot{x}}_{n,n} = \hat{\dot{x}}_{n,n-1} + \beta \left(\frac{z_n - \hat{x}_{n,n-1}}{\Delta t}\right)$$

$$\hat{\ddot{x}}_{n,n} = \hat{\ddot{x}}_{n,n-1} + \gamma \left(\frac{z_n - \hat{x}_{n,n-1}}{0.5\Delta t^2}\right)$$

当γ = 0的时候，退化成我们上一节讲的α − β滤波器

α − β − (γ)滤波器总结

α − β − γ滤波器有很多种类，但都是基于相同的原理：

  • 当前状态估计基于状态更新方程(state update equations)计算的
  • 下一阶段的状态估计则基于动态模型(dynamic model equations)计算的

这些滤波器之间的区别在于参数α − β − γ的选择。一些滤波器倾向于使用固定的三元组参数，但另一些则倾向于在每次迭代的时候重新计算参数的权重。因此，重点在于如何选择参数。而另一个重要的部分是如何初始化。

一维卡尔曼滤波器

现在我们继续学习卡尔曼滤波器在一唯下的表现。首先我们考虑一些简单模型，比如不带过程噪声的一维卡尔曼滤波器

卡尔曼滤波基于5个方程，我们已经学习了其中的2个，即状态更新方程和动态模型方程。在这个章节我们将直接推导剩下的3个方程，并修正状态更新方程。

和上节的α − β − γ滤波器类似，卡尔曼滤波器也采用测量值-当前状态估计以及下一阶段的状态估计值（预测值）都视为服从正态分布的随机变量。这些随机变量通过均值和方差描述。

下面是卡尔曼滤波器原理示意图

系统接收2个输入，首先是测量值（每个周期传感器测到的值），还有初始化（第一次传入的估计值）。卡尔曼滤波器接收这两个参数，更新步骤结合上一步的预测值和当前的测量值进行状态更新，然后进入预测步骤，输出下一哥状态预测值作为下一个周期的预测值给更新步骤用。最后输出当前最佳的状态计算值

作为随机变量的估计值

估计值和实际值之间的差我们成为估计误差，随着更多的测量，这个误差将逐渐的收敛到0，同时估计值也会靠近真实值，我们无法准确的知道这个误差是多少，但我们可以对这个状态的不确定性做估计

而我们使用p代表状态估计方差(state estimate variance)

作为随机变量的测量值

由于测量误差是随机的，我们可以用σ2（方差）描述，测量误差的标准差即为测量不确定，我们使用r表示测量方差

状态预测

我们接触的第一个例子是测量金条的重量，其中动态系统是常数，即

x̂n + 1, n = x̂n, n

而在第二个例子，我们利用运动方程，将当前状态（目标的位置和速度）预测到了下一个阶段的状态。

$$ \begin{aligned} \hat x_{n+1,n} &= \hat x_{n,n} + \Delta t\hat{\dot{x}}_{n,n}\\ \hat{\dot{x}}_{n+1,n} &= \hat{\dot{x}}_{n,n} \end{aligned} $$ 也就是说，预测的位置相当于当前估计的位置加上估计值的速度与时间的乘积。则预测的速度等于当前的速度估计值。（基于匀速假设）

这个动力模型方程依赖系统。

由于卡尔曼滤波将估计值视为一个随机变量，因此我们必须将估计方差(pn, n)外推到下一个阶段的状态

由于金条问题是静态系统，不确定性将不改变，因此p̂n + 1, n = p̂n, n

其中p是金条重量的估计方差

若是飞机追踪问题，那么预测不确定方程如下：

$$ \begin{aligned} p^x_{n+1,n} =& p^x_{n,n}+ \Delta t^2\cdot p^v_{n,n}\\ p^v_{n+1,n} =& p^v_{n,n} \end{aligned} $$

其中px是位置的估计方差，pv是速度的估计方差。

他们和状态方程类似，这是因为误差随着状态的漂移而漂移。或者更进一步预测不确定性方程是作用在预测值上的一个同态函数。这个同态映射是状态方程。

上面两个例子的意思是，在速度恒定模型上，位置估计的预测方差等于当前的位置预测方差，加上当前的速度估计方差乘以时间的平方。而速度估计的预测方差直接等于当前的速度估计方差。

对任意方差为σ2的正态分布随机变量x，kx也服从正态分布并且方差变为k2σ2。因此在不确定性外推方程上，时间项是平方形式。这个不确定方程也称为协方差外推方程..不过解释这个不是现在要做的事情。

状态更新

为了估计系统的当前状态，我们结合了两个随机变量。 + 先验状态估计（在前一个状态预测当前状态的估计值） + 测量值

卡尔曼是最优滤波器，因为其结合先验状态和测量值。而当前状态估计正是测量值和先验状态的加权平均。

$$ \begin{aligned} \hat x_{n,n} = &w_1z_n + w_2\hat x_{n,n-1}\\ w_1+w_2 =& 1 \end{aligned} $$

那么把w2 = 1 − w1带入方程并转化为方差关系式就有 $$ \begin{aligned} p_{n,n} = w_1^2r_n + (1-w_1)^2p_{n,n-1} \end{aligned} $$

其中 pn, n是最优组合估计值的方差。 pn, n − 1是先验估计值x̂n, n − 1的方差 rn是测量值zn的方差。

注意方差的公式，对随机变量kx，则方差就是k2σ2。

我们不喜欢不稳定的值，为此我们尽可能的希望pn, n是最小的，基于现有关系式，我们希望通过调整w1使得pn, n最小，那么我们对w1求导并令导数为0.有

$$ \frac{dp_{n,n}}{dw_1} = 2w_1r_n - 2(1-w_1)p_{n,n-1} = 0 $$

我们可以解出

$$ w_1 = \frac{p_{n,n-1}}{p_{n,n-1}+r_n} $$

那么我们将这个结果应用到估计方程x̂n, n上

x̂n, n = w1zn + (1 − w1)x̂n, n − 1 = w1zn + x̂n, n − 1 − w1x̂n, n − 1 = x̂n, n − 1 + w1(zn − x̂n, n − 1)

那么完整的状态更新方程如下：

$$ \begin{aligned} \hat x_{n,n} = \hat x_{n,n-1} + \frac{p_{n,n-1}}{p_{n,n-1} + r_n}(z_n - \hat x_{n,n-1}) \end{aligned} $$

这和我们一开始的α − β − (γ)滤波器特别像，不过这里的创新点是w1的极值应用在状态方程中，但这个w1我们叫卡尔曼增益，记为Kn，而卡尔曼增益是一个介于0到1的数。

最后，我们需要计算当前状态估计的方差，我们已经知道，方差之间的关系由以下公式给出，注意，一维协方差其实就是方差，

pn, n = (1 − Kn)pn, n − 1 其中Kn是卡尔曼增益

这个方程告诉我们，随着滤波器的每次迭代，估计的不确定性在降低，当测量噪声较高时，卡尔曼增益就会非常低，反过来一样，若噪声较低，那么卡尔曼增益将会很高，不确定度也会迅速向0收敛

我们来自己测量一下，假设我们在测量一栋建筑物的高度，并且期望的精度是3cm(标准差 = 3cm)，那么我们应该持续测量直到状态估计方差σ2小于9cm

算法设计

我们来联系之前学到的所有东西。

滤波器输入的参数有：

  • 初始化
    • 初始化只需要运行一次，且它提供两个参数：
      • 初始系统状态(x̂0, 0)
      • 初始系统方差(p0, 0) 初始化参数可以由其他系统提供，其他过程，或是通过经验猜出的值。即便初始化参数不够准确，卡尔曼滤波其实也能够收敛到真实值。
  • 测量值
    • 测量值在每个滤波器的周期中会测量一次，并且提供2个参数
      • 测量系统状态zn
      • 测量方差rn

而滤波器的输出有：

  • 系统状态估计x̂n, n
  • 系统方差pn, n

注意的是，上述过程不含过程噪声，这是我们接下来研究的重点对象。状态外推方程和协方差外推方程取决于系统的动力学特征。

算法如下设计：

  • 步骤0：初始化 系统提供两个参数，并运行一次

    • 系统初始状态x̂0, 0
    • 初始状态方差p0, 0

  • 步骤1 测量

    • 系统测量状态zn
    • 测量方差rn

  • 步骤2：系统更新 状态更新过程负责对系统的当前状态做状态估计 输入有：

    • 测量值(zn)
    • 测量方差rn
    • 先验系统状态估计x̂n, n − 1
    • 先验系统状态估计方差pn, n − 1

    基于这些输入，状态更新过程计算卡尔曼增益并给出两个输出

    • 当前系统状态估计值和估计方差x̂n, n pn, n

  • 步骤3： 预测

    预测过程基于系统的动力学模型，将当前系统状态估计值及其方差外推至下一个系统状态。 在滤波器的第一次迭代中，初始化参数被视为先验状态估计值和方差。在随后的滤波器迭代中，预测输出将用作先验（预测的）状态估计值和方差。

卡尔曼增益带来的直觉

让我们重写状态更新方程为

x̂n, n = x̂n, n − 1 + Kn(zn − x̂n, n − 1) = (1 − Kn)x̂n, n − 1 + Knzn

那么我们可以看到，卡尔曼增益Kn是测量值的权重，则1 − Kn则是当前估计值的权重。这意味着若Kn趋向于0的时候，则表明测量值越不稳定且估计的不确定性很低。因此我们将给估计值很高的权重，给测量值低权重。一种特殊情况是，若测量不确定性的值等于估计的不确定性，那么Kn = 0.5

而卡尔曼增益定义在形成新估计值的时候，测量值和先验估计值的权重，告诉我们测量值对最终估计值的修正程度。

测量高层建筑

现在我们来使用卡尔曼滤波器对一座高层建筑进行估计

该模型的高度在一段时间内不会有变化，我们假设它满足这些参数：

  • 建筑物的真实高度是50米
  • 高度计的测量（标准差）在5米内
  • 十次测量值分别是49.03, 48.44, 55.21, 49.98, 50.6, 52.61, 45.87, 42.64, 48.26, 55.84

初始化

我们假设第一次估计值是x̂0, 0 = 60m，现在，我们初始化估计方差，人类的估计误差估计在15m，那么σ = 15, σ2 = 225。那么p0, 0 = 225

因为一开始我们还没测量，则我们对下一个时刻的估计就是上面的两个参数。因此x̂1, 0 = 60m，且p1, 0 = 225。那么我们开始第一次测量

第一次测量

首先，第一次测量为z1 = 49.03m，方差r1 = 25

现在我们计算卡尔曼增益

更新

$$ \begin{aligned} K_1 = \frac{p_{1,0}}{p_{1,0}+r_1} = \frac{p_{1,0}}{p_{1,0}+25} = 0.9 \end{aligned} $$

当前估计值为

x̂1, 1 = x̂1, 0 + K1(z1 − x̂1, 0) = 60 + 0.9(49.03 − 60) = 50.13

当前估计方差为

p1, 1 = (1 − K1)p1, 0 = (1 − 0.9)225 = 22.5

然后我们把这些作为下一个时刻的先验估计值提供，记为

$$ \begin{aligned} \hat x_{2,1} = \hat x_{1,1} = 50.13\\ p_{2,1} = p_{1,1} = 22.5 \end{aligned} $$

第二次更新为

$$ \begin{array}{rcl} K_2 &=& \frac{p_{2,1}}{p_{2,1}+r_2} = \frac{22.5}{22.5+25} = 0.47\\ \hat x_{2,2} &=& \hat x_{2,1} + K_2(z_2 - x_{2,1}) = 50.13+0.47(48.44-50.13) = 49.33\\ p_{2,2} &=& (1-K_2)p_{2,1} = (1-0.47)22.5 = 11.84 \end{array} $$
